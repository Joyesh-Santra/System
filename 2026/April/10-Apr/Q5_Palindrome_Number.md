## Palindrome Number

Determine if an integer is a palindrome (reads the same forwards and backwards).

**Input:** A single integer `n`.

**Output:** Print `YES` if palindrome, else `NO`.

**Constraints:** `-2^31 ≤ n ≤ 2^31 - 1`


### 💡 Examples

**Example 1:**
- **Input:** `121`
- **Output:** `YES`

**Example 2:**
- **Input:** `-121`
- **Output:** `NO`
- **Explanation:** Negative numbers are not palindromes
